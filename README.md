# To-Do List Application  
This is a simple *To-Do List Application* developed as part of the CodSoft Internship Program (Task 1). The application allows users to efficiently manage their tasks using a menu-driven interface.  

---------------------------------------------------------------------------------------------

## Features  
- *Add a Task:* Add a new task to your list.  
- *View Tasks:* Display all the tasks in the list.  
- *Update a Task:* Modify the description of an existing task.  
- *Mark Task as Completed:* Mark any task as completed.  
- *Delete a Task:* Remove a task from the list.  
  
-------------------------------------------------------------------------------------------------

## Technologies Used  
- *Programming Language:* Python  
- *IDE:* PyCharm
